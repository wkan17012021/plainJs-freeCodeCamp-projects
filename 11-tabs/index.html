Intro

This app is useful as it shows how to connect different groups of elements styling changes triggered by JS. When the tab button is clicked, a style changes occurs across all three buttons. Subsequently, the correct content should also be shown.

Learning Points

The app demonstrates the use of e.target. In bubbling, the event is first triggered on the element that was clicked, and then it bubbles up through its ancestors (parents, grandparents, etc.) until it reaches the root element. target references the element that triggered the event. Note this is subtly different from e.currentTarget, which will always reference what element had the eventlistener attached to - this is your choice what the currentTarget could be.

Difficult to maintain

Requires hard coding markup: section content ids need to match button tab ids, should the project grow.
