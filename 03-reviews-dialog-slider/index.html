Managed to complete this mini-app independently i.e without watching the video guide. It helps that the previous two projects builds on this one though

My method

Grab a node list and iterate through them using forEach to add event listeners to the forwards and backwards buttons.

Use event.currentTarget to ascertain the button that was clicked and append classList.contains to reveal the list of classnames and query if the specific class exists.

Used conditionals to increment or decrement a counter variable which also served as the data array index position.

Cleaned up the code to have a function that sets the image element attributes and textContent by feeding the counter variable as the data array index position.

Improvements

There is still some very similar if/else conditionals - perhaps these could be encapsulated.

When you click the Surprise Me button, you could be returned the same person if the randomly generated ID is duplicated. In the color flipper app, we used two flag variables to compare previous Index with new Index. If they are the same, rerun the randomiser function. This seems like the same solution.
