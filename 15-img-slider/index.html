How does the slider work?

  • Initial state of images are laid on top of each other using position absolute. The div container is position relative.
  • Use overflow hidden to maintain a view area only around the slider container.
  • Initially get all items in a row using style left property.
  • When we click on the buttons, we use transform X property to move the image left or right.
  • Take a look at the setup of the JS for comments
  • Use a count variable to keep track of the index position. Use it to invoke the function controlling the actual carousel movement.
  • The carousel movement uses the forEach to grab all items and move them either right or left using style.transform = template literal for translateX(count var value multipled by 100%)
  • Setup if conditionals to handle how the slider should behave when you get to the end of the list of items or try to click on previous button when you are already at the start of the image list.
